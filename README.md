# Theme
### Dependencies
* TYPO3 7.6.0 - 8.7.99
* vhs 3.0.0 - 5.0.99
## Description
This Extension adds a base theme for a fresh TYPO3 instance.
### How to install?
* Get and install [yarn](https://yarnpkg.com/lang/en/docs/install/)
* Navigate to the root of the extension and run:
````bash
yarn install
````
